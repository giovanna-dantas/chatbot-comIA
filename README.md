# Chatbot com Inteligência Artificial

Aplicação de chatbot desenvolvida em **Python** utilizando **Streamlit** e integração com **LLM via API**, permitindo interação em linguagem natural com respostas geradas por inteligência artificial.

## Objetivo

Demonstrar a integração entre **interfaces web em Python** e **modelos de linguagem**, criando uma aplicação interativa de conversação.

## Tecnologias

- Python
- Streamlit
- LLM (Llama via Groq API)
- python-dotenv

## Principais funcionalidades

- Interface de chat interativa
- Comunicação com modelo de IA
- Histórico de mensagens durante a sessão
- Respostas geradas em tempo real

---

## Aprendizados aplicados

- Integração com APIs de IA
- Desenvolvimento de interfaces com Streamlit
- Gerenciamento de estado em aplicações interativas
